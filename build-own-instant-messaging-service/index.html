搭建自己的即时聊天服务

哪个男孩不想拥有像QQ一样的聊天软件呢？

前期准备

  • Openfire
  • Pidgin

Openfire Linux 安装

Openfire 配置

安装完成后访问 URL:9090 进入设置界面

在 域 和 Server Host Name 填入域名/IP

设置完成后进入管理界面

前往 插件 - 插件管理 - 有效的插件

找到 Registration 安装注册插件

前往 用户/用户群 - 用户 - Registration Properties
勾选 Enable users to register via a web page at... 开启用户通过网页注册功能，保存设置

用户访问 URL:9090/plugins/registration/sign-up.jsp 就可以进入注册页面

Pidgin 配置

点击 添加 按钮添加账号

选择 XMPP协议，填写其余登录信息后点击 添加

点击 好友 - 新即时消息 填入名称可以直接给对方发送消息

也可以双方互加好友后发送消息

完成

搭建自己的即时聊天服务
